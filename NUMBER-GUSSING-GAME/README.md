# Number Guessing Game

## Project Overview
This project is a simple command-line Number Guessing Game built using Python. The program randomly generates a number, and the user has to guess it. After each guess, the program provides hints indicating whether the guessed number is too high or too low until the correct number is guessed.

## Features
- Random number generation
- User-friendly command-line interface
- Hints for higher or lower guesses
- Unlimited attempts (or limited attempts if implemented)
- Displays a success message when the correct number is guessed

## Technologies
- Python
- Random Module

## Workflow
- Generate a random number
- Ask the user to enter a guess
- Compare the guess with the generated number
- Display hints (Too High / Too Low)
- Repeat until the correct number is guessed
- Display a congratulatory message

## Python Concepts Used
- Variables
- User Input (`input()`)
- Conditional Statements (`if-elif-else`)
- Loops (`while`)
- Random Module (`random`)
- Comparison Operators

## Output
The program allows the user to guess a randomly generated number by providing hints after each incorrect guess and ends when the correct number is guessed.
