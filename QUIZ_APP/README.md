# Quiz App

## Project Overview
This project is a simple command-line Quiz Application built using Python. It presents multiple-choice questions to the user, checks the answers, calculates the score, and displays the final result at the end of the quiz.

## Features
- Multiple-choice questions
- User-friendly command-line interface
- Automatic answer validation
- Score calculation
- Displays final result
- Easy to add or modify questions

## Technologies
- Python

## Workflow
- Display quiz questions
- Accept the user's answer
- Check whether the answer is correct
- Update the score
- Move to the next question
- Display the final score after all questions are completed

## Python Concepts Used
- Variables
- Lists
- Dictionaries (if implemented)
- User Input (`input()`)
- Conditional Statements (`if-elif-else`)
- Loops (`for` / `while`)
- Functions

## Output
The program asks the user a series of quiz questions, evaluates each answer, keeps track of the score, and displays the final score at the end of the quiz.
