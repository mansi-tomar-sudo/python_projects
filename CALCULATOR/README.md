# Python Calculator

## Project Overview
This project is a simple command-line calculator built using Python. It performs basic arithmetic operations based on the user's input and displays the result.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- User-friendly menu
- Multiple calculations (if implemented)

## Technologies
- Python

## Workflow
- Display calculator menu
- Take user input
- Select arithmetic operation
- Perform calculation
- Display result
- Repeat until user exits

## Python Concepts Used
- Variables
- User Input (`input()`)
- Functions
- Conditional Statements (`if-elif-else`)
- Loops (`while`)
- Arithmetic Operators

## Output
The program performs the selected arithmetic operation and displays the calculated result on the console.
