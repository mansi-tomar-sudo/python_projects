# Python Mini Projects Collection

## Project Overview
This repository contains a collection of beginner-friendly Python mini projects. Each project is designed to help understand core Python concepts such as variables, loops, functions, conditional statements, file handling, modules, API integration, and object-oriented programming. These projects provide hands-on practice and strengthen problem-solving skills through real-world applications.

## Project Files

### 📌 calculator.py
A command-line calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, division, and modulus.

### 📌 expenses_tracker.py
An expense tracking application that allows users to add and view daily expenses. The expense records are stored in `expenses.txt`.

### 📌 expenses.txt
Stores expense records created by the Expense Tracker application.

### 📌 number_guessing_game.py
A game where the computer randomly generates a number and the user tries to guess it with the help of hints.

### 📌 password_generator.py
Generates strong and random passwords based on the user-specified password length.

### 📌 quiz_app.py
A command-line quiz application that asks multiple-choice questions, checks answers, and calculates the final score.

### 📌 student.py
A Student Management System that allows users to add, view, search, update, and delete student records.

### 📌 student.txt
Stores student records used by the Student Management System.

### 📌 todo_list.py
A simple To-Do List application for adding, viewing, updating, and deleting daily tasks.

### 📌 weather.py
A weather application that fetches real-time weather information for any city using the OpenWeatherMap API.

### 📌 cwd_finder.py
A simple utility program that displays the current working directory using Python's built-in `os` module.
