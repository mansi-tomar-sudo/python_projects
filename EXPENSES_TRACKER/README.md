# Expense Tracker

## Project Overview
This project is a simple command-line Expense Tracker built using Python. It allows users to add, view, and manage their daily expenses. The expense records are stored in a text file, making the data persistent even after the program is closed.

## Files

### expenses_tracker.py
This is the main Python file that contains the complete program logic. It handles user input, records expenses, displays saved expenses, and manages file operations.

### expenses.txt
This text file stores all expense records. Each time a new expense is added, it is saved in this file so that the data can be accessed later.

## Features
- Add new expenses
- View all expenses
- Store expense records permanently
- Simple menu-driven interface
- Easy to use

## Technologies
- Python
- File Handling

## Workflow
- Display menu
- Accept user choice
- Add expense details
- Save data to `expenses.txt`
- Read and display saved expenses
- Repeat until user exits

## Python Concepts Used
- Variables
- Functions
- Loops
- Conditional Statements (`if-elif-else`)
- User Input (`input()`)
- File Handling
- String Manipulation

## Output
The program stores expense records in `expenses.txt` and allows users to view their saved expenses through the command-line interface.
