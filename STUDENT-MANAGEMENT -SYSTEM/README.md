# Student Management System

## Project Overview
This project is a simple command-line Student Management System built using Python. It allows users to add, view, search, update, and delete student records. The student data is stored in a text file, ensuring that records are saved even after the program is closed.

## Files

### student.py
This is the main Python file that contains the complete program logic. It manages student records, processes user input, performs CRUD (Create, Read, Update, Delete) operations, and handles file operations.

### student.txt
This text file stores all student records. It acts as the project's database, allowing the program to save and retrieve student information whenever required.

## Features
- Add new student records
- View all student records
- Search for a student
- Update student details
- Delete student records
- Store records permanently using a text file
- Simple menu-driven interface

## Technologies
- Python
- File Handling

## Workflow
- Display the main menu
- Accept the user's choice
- Perform the selected operation
- Save or retrieve data from `student.txt`
- Display the updated information
- Repeat until the user exits

## Python Concepts Used
- Variables
- Functions
- Lists
- User Input (`input()`)
- Conditional Statements (`if-elif-else`)
- Loops (`while`)
- File Handling
- String Manipulation

## Output
The program manages student records efficiently by storing data in `student.txt` and allows users to add, view, search, update, and delete student information through the command-line interface.
