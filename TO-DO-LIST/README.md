# To-Do List Application

## Project Overview
This project is a simple command-line To-Do List application built using Python. It helps users organize and manage their daily tasks by allowing them to add, view, update, and delete tasks. The application provides an easy way to keep track of pending work through a menu-driven interface.

## Features
- Add new tasks
- View all tasks
- Update existing tasks
- Delete completed or unwanted tasks
- Simple and user-friendly interface
- Easy task management

## Technologies
- Python

## Workflow
- Display the main menu
- Accept the user's choice
- Add, view, update, or delete tasks
- Display the updated task list
- Repeat until the user chooses to exit

## Python Concepts Used
- Variables
- Lists
- Functions
- User Input (`input()`)
- Conditional Statements (`if-elif-else`)
- Loops (`while`)
- List Operations

## Output
The program allows users to efficiently manage their daily tasks by adding, viewing, updating, and deleting tasks through a command-line interface.
