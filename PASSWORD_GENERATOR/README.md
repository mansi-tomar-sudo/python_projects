# Password Generator

## Project Overview
This project is a simple Password Generator built using Python. It generates strong and random passwords based on the user's desired password length. The generated passwords contain a combination of uppercase letters, lowercase letters, numbers, and special characters to improve security.

## Features
- Generate random passwords
- User-defined password length
- Includes uppercase and lowercase letters
- Includes numbers
- Includes special characters
- Fast and easy to use

## Technologies
- Python
- Random Module
- String Module

## Workflow
- Ask the user for the desired password length
- Create a pool of letters, numbers, and special characters
- Randomly select characters
- Generate the password
- Display the generated password

## Python Concepts Used
- Variables
- User Input (`input()`)
- Functions
- Loops
- Random Module (`random`)
- String Module (`string`)
- String Manipulation

## Output
The program generates a secure random password based on the user's specified length and displays it on the console.
